# New AI Platform HTML

HTML dashboard frontend for the NStarx AI platform.

## Overview

A static HTML frontend providing a web dashboard for an AI platform. Includes pages for login, registration, user profiles, AI zone management, deployment tracking, and data tables.

## Pages

- `index.html` - Main dashboard landing page
- `login.html` - User authentication page
- `register.html` - New user registration
- `profile.html` - User profile management
- `ai-zones.html` - AI zone configuration and monitoring
- `deployments.html` - Deployment status and management
- `table.html` - Data table view
- `assets/` - Static assets (CSS, JS, images)

## Usage

Open `index.html` in a browser or deploy to any static web server.
